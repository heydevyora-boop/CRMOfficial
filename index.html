MRP Auto-Markup Settings
When enabled, MRP is automatically calculated as: MRP = Rate × (1 + Markup%)
This fills the MRP field whenever a rate is applied from the calculator. The telecaller/client sees the MRP — your actual rate stays internal.

📐 Column Formulas — Applied to Every Quotation
These formulas run automatically on every row in every quotation. Change once — updates all future quotations instantly. Like Excel cell formulas but for the entire CRM.
MRP Column Formula
How MRP is calculated from Rate. Default: MRP = Rate × 1.125 (12.5% above rate)
Variables: rate, qty, pct
Preview: Rate Rs.350 → MRP Rs.393.75
Amount Column Formula
How Amount is calculated. Default: Amount = Qty × Rate
Variables: rate, qty, mrp
Preview: Qty 4, Rate Rs.500 → Amount Rs.2000
💡 How it works: When you type a Rate in any quotation row, the MRP and Amount columns auto-fill using these formulas. Change the formula here once → applies to all quotations from that point forward. You can still manually type in any cell to override.
